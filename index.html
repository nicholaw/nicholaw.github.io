Appointment Scheduler

This project was my final assessment submission for Western Governors University's C195: Advanced Java Concepts course. This application allows a user to add or edit customers to the database of a hypothetical international corporation and edit or schedule appointments for those customers. The application can also generate reports regarding how many appointments of each type are scheduled for each month. The hypothetical corporation has offices in Arizona, New York, Montreal, and the United Kingdom, and therefore offers services in English and French across multiple time zones.

  • Languages: Java, MySQL
  • Tools: JavaFx, CSS, Eclipse
  • GitHub Repository

Delivery Truck Routing

This project served as my final assessment for Western Governors University's C950: Algorithms and Datastructures II. The application takes a list of packages as well as a list of possible delivery locations and determines an order in which to deliver the packages while keeping the total distance driven by the delivery trucks under 140 miles.

  • Languages: Python
  • Tools: XML, Notepad++
  • GitHub Repository

Keep Talking and Nobody Explodes

This was a group project for Salt Lake Community College's CSIS 2810: Computer Architecture Class. For the project, I and three other students used Arduinos and electrical components to create a physical representation of the digital party game "Keep Talking and Nobody Explodes" by Steel Crate Games. Each Arduino controlled a discrete puzzle which had to be solved to defuse the 'bomb,' and an additional 'master' arduio controlled the game. My primary engineering contribution to this project was the combination password puzzle.

  • Languages: C++
  • Tools: Arduino Uno, Arduino IDE
